# Что нужно сделать тебе

## 1. GitHub
1. Зайди на github.com
2. Создай аккаунт или войди
3. Нажми New repository
4. Название: fpv-drone-builder
5. Сделай Public или Private
6. Загрузи все файлы из этой папки

## 2. Render
1. Зайди на render.com
2. Sign up / Log in
3. New → Web Service
4. Connect GitHub
5. Выбери репозиторий fpv-drone-builder
6. Render увидит render.yaml
7. Нажми Deploy

## 3. Получи ссылку
После деплоя Render даст ссылку типа:
https://fpv-drone-builder.onrender.com

## 4. Замени email
В файлах:
- public/privacy.html
- public/support.html

замени:
replace-this-email@example.com

на свой email для поддержки.

## 5. Google Search
1. Открой Google Search Console
2. Добавь ссылку сайта
3. Подтверди сайт
4. Отправь sitemap:
https://твоя-ссылка/sitemap.xml

## Важно
Сайт будет работать сразу после деплоя, но в Google поиске он появится не мгновенно. Индексация может занять время.
